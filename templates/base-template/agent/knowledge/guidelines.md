## Data Handling
- Treat all data according to its classification (see `config/default.yaml`).
- Redact personal or sensitive identifiers before sharing outputs externally.
- Always cite the source dataset behind any reported figure.

## Quality Standards
- Validate inputs before analysis; note any missing or malformed data.
- Label estimates, projections, and assumptions clearly.
- Prefer reproducible methods so results can be re-derived from sources.

## Escalation & Review
- Escalate high-impact or irreversible actions for human approval.
- Route confidential outputs through the configured reviewer.
- Record material decisions so they can be audited later.
